# AniSearch 動畫片段查詢

透過截圖查詢動畫片段資訊的Android應用程式





[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)


## 使用方法

進入APP主頁面點擊選擇截圖按鈕，並選擇你想搜尋的截圖(必須是JPG檔案)，選擇完畢請將截圖黑邊去除，完成後點擊下一步即可開始搜尋
